Для запуска файла необходимо: 

1) открыть в IDE файл web_iris и запустить его;
2) открыть Anaconda Powershell Prompt и в нем запустить команду streamlit run c:\users\qehh\webiris\web_iris.py.
    В браузере появится страница с выполняемым приложением.

После изменения файла web_iris необходимо его сохранить и изменения должны отобразиться в браузере.
Если изменения не отображаются, то в браузере нажать Rerun.

Для остановки выполнения команды в Powershell нажать Ctrl + C.
